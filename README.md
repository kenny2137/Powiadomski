# 📳 Powiadomski

### Description
The Powiadomski project uses GSM modules for automatic SMS notification management, helping to authenticate remotely on various platforms. The project arose from the need to test Arduino UNO Q and centralize the collection of text messages and forward them to a messenger.

Stuff used in this project:

- Arduino Uno Q (Brain of this project)
- Chinese SIMCOM 7670E Module
- Cables to connect Arduino to the GSM module with UART
